# Portafolio Desarrollador — Miguel Tamat

Portafolio profesional desarrollado con React, Vite y Tailwind CSS.

## Stack

- React 19
- Vite 6
- Tailwind CSS 4

## Estructura

```
src/
├── assets/          # Imágenes e iconos locales
├── components/      # Componentes reutilizables
├── data/            # portfolioData.js — fuente de datos centralizada
├── layouts/         # Sidebar y layout principal
└── pages/           # Secciones del portafolio
```

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Personalización

Edita `src/data/portfolioData.js` para actualizar tu información personal, habilidades, proyectos y enlaces.
